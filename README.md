# Users

# GET /users
Get all users.

# GET /users/:id
Get a user with particular id.

# POST /users
Creates a user.

# PUT /users/:id
Updates a user with particular id.

# DELETE /users/:id
Deletes a user with particular id.

# GET /users/:id/subscribers
Get all subscribers for a user with particular id.
